# Jonathan Martin's Tap

My custom Tap for things specific to me or things I find annoying in homebrew-core

## How do I install these formulae?

`brew install jonathanmartin-dev/tap/<formula>`

Or `brew tap jonathanmartin-dev/tap` and then `brew install <formula>`.

## Currently available formula

* Terragrunt

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
